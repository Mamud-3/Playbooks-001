# Install tools on ubuntu workstation

## Download ansible

make requirements

## VSCode

make vscode

## Terminator

make terminator

## Terraform

make terraform

## awscli

make awscli

## docker & docker-compose

make docker

## helm

make helm

## kubectl

make kubectl

## postman

make postman

## hyper

make hyper

## requirements

python : `sudo apt install python`<br>
pip : `sudo apt install python-pip`<br>
ansible : `sudo pip install ansible`
